XED_DLL_EXPORT xed_cpuid_bit_enum_t xed_get_cpuid_bit_for_isa_set ( xed_isa_set_enum_t  isaset,
xed_uint_t  i 
)

Returns the name of the i'th cpuid bit associated with this isa-set.

Call this repeatedly, with 0 <= i < XED_MAX_CPUID_BITS_PER_ISA_SET. Give up when i == XED_MAX_CPUID_BITS_PER_ISA_SET or the return value is XED_CPUID_BIT_INVALID.
